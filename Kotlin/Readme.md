### 코틀린이란?
- 젯 브레인이라는 회사에서 만든 언어(통합개발 환경 툴)이다. 
- 안드로이드 스튜디오는 인텔리제인을 기반으로 만들어졌다.
    - 코틀린은 자바를 보완하기 위해 만들어진 언어
    - 자바로 개발할 수 있는 언어는 다 코틀린으로 개발할 수 있음
- google이 코틀린 언어를 공식 언어로 채택함
    - 왜? -> 자바 언어보단 사용하기 편하고 쉽기 때문에
